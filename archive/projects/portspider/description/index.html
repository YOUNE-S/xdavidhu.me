Description:

A framework for scanning huge networks for specific ports, services and devices. portSpider works by using modules with a Metasploit Framework like interface. All of the modules created by me use multithreading, which means that the scans are extremely fast. (some cases even faster than the "standard" network scanner, Nmap) Moreover the project is make it easy for anyone to contribute and create new modules for the framework. Modules available by the time of writing: http, mongodb, mysql, ssh, printer, gameserver, manual
